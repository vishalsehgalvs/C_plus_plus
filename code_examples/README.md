# C++ Code Examples

Three curated sets of C++ code, organized by complexity and purpose.

---

## Structure

```
code_examples/
├── basic/           — Beginner-friendly examples, one concept per file
├── advanced/        — Intermediate to advanced C++ features
└── leetcode_style/  — Competitive programming / interview problems
```
